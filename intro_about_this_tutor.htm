本教材將主要以應用及實作的角度，介紹音訊分析的一些大題目及其作法，以及機器學習方面的一些先備知識。使用本教材的讀者，至少必須對國高中的物理還有點印象、學過或瞭解計算機概論，以及會寫任意一種程式語言，如果是 C 或 Python 的話最好。再者，雖然本教材預計不會提到太高深的數學，但各位可能至少需要對線性代數、微積分及機率有些概念，否則會難以理解演算法背後的原理的奧妙。另外，為了鼓勵各為自己執行看看範例，所以若能經由範例程式產生出的圖片或音訊等資料，原則上不會一併展示。
